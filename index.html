Press/Click Any Key to Start

Note: You can also use arrow keys or a, w, s, d keys on keyboard to select the keyboard.
How to Play
  1. At every level, a new button will blink.
  2. You need to remember the order of the blinking button.
  3. Click on buttons in the same order as shown in previous level and add the newly blinking button to that order.
  4. If the order in which you click is wrong, the game will end.
